# infomation-tips
Display system operation information on the screen

## Installation

* mkdir build
* cd build
* cmake .. 
* make
* ./infomation-tips

## Build deb or tar.gz

* mkdir build
* cd build
* cmake .. ***or*** cmake .. -DBUILD_TYPE=TGZ
* make packae

*You get the file "\*. deb" or "\*.tar.gz"*
